# Wizja produktu

## Czym jest poe1-trainer

`poe1-trainer` to aplikacja, która pomaga graczowi coraz lepiej rozumieć Path of Exile 1 podczas realnego grania konkretnym buildem.

Nie ma to być system, który podejmuje wszystko za użytkownika. Ma raczej działać jak warstwa treningowa:

- pokazuje, że właśnie teraz warto podjąć decyzję,
- przypomina o rzeczach, które doświadczeni gracze sprawdzają automatycznie,
- tłumaczy, dlaczego dany moment jest ważny,
- uczy powtarzalnych wzorców i skrótów myślowych.

## Problem, który rozwiązujemy

Początkujący i średniozaawansowany gracz często:

- nie wie, kiedy powinien zmienić setup,
- nie zauważa, że właśnie odblokował ważny gem lub craft,
- nie rozumie, dlaczego run zaczyna zwalniać,
- nie umie ocenić, które decyzje są naprawdę ważne na danym etapie.

Efekt jest taki, że guide bywa tylko ścianą tekstu, a wiedza nie przechodzi w nawyk.

## Obietnica produktu

System ma pomagać graczowi przejść od wykonywania poleceń do rozumienia decyzji.

## Zasady projektowe

- Każda rekomendacja powinna mieć krótki powód.
- Jeśli coś jest niepewne, system powinien to zaznaczyć.
- Jeśli czegoś nie da się ustalić bezpiecznie, system powinien poprosić o potwierdzenie.
- UI ma pokazywać stan progresu i następne sensowne akcje, nie tylko pełny przewodnik.
- Wiedza ekspercka ma być podawana w małych, kontekstowych porcjach.
